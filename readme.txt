8. 執行、使用方式說明：
   主程式：
   編譯完成後，在檔案目錄下會產生名為router的執行檔。
   執行檔的命令格式為：
   ./router <input file name> <output file name>

   例如：要跑mps的第一筆測試資料，並輸出。
   ./router gr4x4.in gr4x4.out
   即可在同目錄下得到gr4x4.out的輸出檔案

9. 執行結果說明（說明執行結果的觀看方法，及解釋各項數據等）：
   執行結果：
	輸出如題目所要求， 分成2個部分：
	  1. 只有兩個數字的一列，即為每一個net id的第一列
	     第一個數字代表net id，第二個數字代表走過的路徑長
	  2. 從第一列之後每一列皆有4個數字。
	     依序由起點到終點的路徑
	     前兩個數字為一個出發點，後兩個數字為到達點
	     由此可判斷出走過的路徑。
	     詳細分析在report當中。
